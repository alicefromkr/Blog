## AWS Cloud School을 선택한 이유

한달 동안 SAP 공부하면서 회계보다는 다른데 더 관심이 갔다.

SAP HANA S/4를 사용하면서 '이건 어떻게 서버를 관리하고 있을까?', 'SAP HANA S/4가 웹버전으로 나오면서 클라우드 이전 할 것같은데 어떻게 했으려나?' 등 인프라 구성이 궁금했다. 인프라 어떻게 구성되었는지 찾아볼까 하면서 이것저것 찾아보다가 [메가존클라우드의 SAP on AWS]("https://www.megazone.com/competency/sap/")를 보게 되었다. AWS 설계를 어떻게 했는지 간략하게 설명는데 정말 하나도 모르겠더라. 더 찾아보니 [삼성 SDS 아모레퍼시픽 SAP 전환 구축 사례]("https://www.samsungsds.com/kr/case-study/amorepacific.html")도 AWS를 이용해서 SAP를 구축했다.[LG CNS의 깨끗한나라 SAP 마이그레이션]("https://amazon-web-services.lgcns.com/case/case-sap-kleannara")을 보면서 요즘 대기업들 주요 관심사 중 하나구나 싶었다.

그러다 2023년 12월에 최종 면접에서 떨어지고, '이제 뭐하지?' 망연자실하면서 자소설을 딱 켰는데 AWS Cloud School 4기 모집 공고글이 보였다. 진짜 타이밍 대박이다 라고 생각하면서 바로 지원했다.

## 24시간의 면접 준비

지원하고 사전 시험을 보고, 어느새 면접이 다가왔다! 아니.. 나 클라우드에 대해 아는게 아무것도 없어서 책부터 펼쳤다. 때마침 밀리의 서재에서 ["클라우드 AWS자격증으로 시작하기"]("https://millie.page.link/VytLN")라는 책이 있어 읽기 시작했다. 당시의 지인들이랑 온라인으로 공부를 했는데, 함께 공부하니깐 더 집중이 잘되고, 여러 사람들의 설명을 들으니 더 잘 머리속에 들어왔다. 덕분에 면접장 가서 열심히 답변 할 수 있었다.

**그날 저와 밤새준 모든 분들께 다시한번 감사드립니다.**

면접에는 AWS 직원분, 네트워크 강사님, 한국전파진흥협회 직원분이 계셨다. 다대다 면접이었고, AWS 직원분과 함께한 면접이 정말 재밌었다. 다양한 시나리오를 제시하셨는데, 그 상황에 대한 솔루션을 면접자들이 답변하는 방식이었다. 새로운 면접 방식이었는데, 티키타카가 재밌었다.
